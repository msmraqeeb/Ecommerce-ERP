# **App Name**: KP ERP

## Core Features:

- Overview Dashboard: Provides a summary of key e-commerce metrics like total sales, order status, and popular products.
- Order Management: Enables viewing, editing, and processing orders, including status updates and shipment tracking.
- Product Management: Allows adding, editing, and removing product listings, including details, images, and pricing, all synchronized with Kidsparadise.com.bd.
- Inventory Management: Automated updates and monitoring of stock levels and sends notifications when stock is low.
- Customer Profile Management: Enables viewing, editing, and managing customer profiles and purchase histories.
- Data Sync Tool: Uses AI to verify that the synchronization with Kidsparadise.com.bd occurs automatically

## Style Guidelines:

- Primary color: Soft purple (#A099FF) for a calming and modern feel.
- Background color: Light gray (#F0F0F5) for a clean and professional look.
- Accent color: Deep purple (#673AB7) for highlighting interactive elements and key information.
- Body text: 'PT Sans', a humanist sans-serif with a modern look.
- Headline font: 'Poppins', a geometric sans-serif for headlines.
- Use minimalist icons with rounded edges for a friendly and modern aesthetic.
- Clean, grid-based layout with clear hierarchy for easy navigation and readability.
- Subtle transitions and animations for user interactions, providing visual feedback without being distracting.
